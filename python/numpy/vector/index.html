    False values, length n

    Action Code Details

    v.ndim == 1

    np.ptp(v) == 0
    Logical type 
    v.dtype == np.bool_
    Float type 
    v.dtype == np.float_
    Integer type 
    v.dtype == np.int_
    Contains nan 
    np.isnan(v).any()
    Contains inf 
    np.isinf(v).any()
    Contains value 
    value in v
    Does not contain value 
    value not in v
    All finite 
    np.isfinite(v).all()
    All elements are equal to value 
    np.all(v == value)
    All elements are equal to value 
    (v == value).all()
    All numerical elements are close to value 
    np.all(np.isclose(v, value))
